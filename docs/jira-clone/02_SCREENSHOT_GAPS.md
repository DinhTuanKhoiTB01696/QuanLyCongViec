## Đã có ảnh tương đối đủ để bắt đầu UI spec

- Global Sidebar / Shell: có ảnh sidebar, topbar và project tab bar trong nhiều màn dark/light.
- Project Summary: có light và dark.
- Board: có light và dark.
- Backlog: có light old; chưa thấy bản dark đầy đủ trong bộ ảnh này.
- Timeline: có light và dark.
- Calendar: có light và dark + unscheduled work panel.
- List: có dark.
- Forms: có light/dark landing, create options, builder.
- Goals: có project tab dark và standalone Goals pages light.
- Development: có light/dark.
- Code: có dark connect tools.
- Archived work items: có dark empty state.
- Docs: có light/dark.
- Reports: có light và old/cropped.
- Dashboard: có list/create/edit/layout/gadgets.
- Issue detail: có full page, metadata panels, subtasks, linked work items, attachments, activity tabs.
- User menu: có dark user dropdown.
- Settings menu: có dark settings dropdown.

## Thiếu hoặc chưa đủ, AI phải hỏi lại ảnh trước khi code

- In-app Search popup đầy đủ: mới thấy search bar/topbar và public website search overlay; chưa đủ popup search trong app.
- Notification popup: chỉ thấy badge chuông ở một số ảnh; chưa có popup notification mở.
- Create work item modal từ nút `Create/Tạo`: thấy nút, chưa có modal tạo work item rõ ràng.
- Login page: chưa thấy login app đầy đủ.
- Forgot password: chưa thấy.
- Register flow đầy đủ: có signup/landing/modal nhưng chưa đủ trạng thái lỗi/loading/success.
- Backlog dark mode đầy đủ: chưa thấy rõ trong bộ ảnh này.
- Reports dark mode đầy đủ: chưa thấy project Reports dark mở riêng; có Summary dark và dashboard edit dark.
- For You app page: chưa có màn For You nội dung chính đủ rõ; chỉ thấy sidebar item.
- Notification states: unread/read/empty/loading chưa có.
- Permission/admin/backend workflow: ảnh không chứng minh, không được tự làm.
