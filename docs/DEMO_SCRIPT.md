# SprintA Demo Script (5-7 phut)

## 1. Van de (30 giay)

Mo dau bang bai toan cong viec rai rac trong Excel, email va Zalo: kho theo doi trang thai, trach nhiem va muc tieu chung.

## 2. Dashboard (45 giay)

Dang nhap `khoi.nguyen@novatech.vn`, vao `SprintA Enterprise Platform`, gioi thieu du lieu project va cac chi so tong quan.

## 3. Work Items (90 giay)

Mo Kanban de xem task theo workflow; chuyen sang List de minh hoa kha nang quet/loc. Mo mot task co san, khong xoa du lieu demo.

## 4. Workflow va Custom Fields (45 giay)

Mo Project Settings, gioi thieu status dong va bon custom fields. Neu demo thay doi, chuan bi phuong an khoi phuc data truoc.

## 5. Goals (45 giay)

Mo Goals, chon mot goal co san, trinh bay owner, status va progress lien ket thuc thi.

## 6. Integration Hub / Unified Inbox (45 giay)

Mo Integration Hub, cho thay inbox demo va bo loc. Nhac ro Google/Gmail/Slack chi connected khi co OAuth that; GitHub/Zalo sap ra mat khong gia lap ket noi.

## 7. Global AI Copilot (60 giay)

Tai Work Items, mo mot AI widget duy nhat va hoi `Tom tat tinh hinh du an nay`. Giai thich request dung project/workspace context, response doc tu `response.data.data`, `actions` rong va khong fake answer khi provider loi.

## 8. Mobile/PWA (30 giay)

Gioi thieu PWA app shell, manifest va service worker. Khong tuyen bo offline data sync; chi static app shell duoc precache.

## 9. Ket luan (20 giay)

SprintA gom quan ly cong viec, muc tieu, bao cao, inbox va AI trong mot workspace, voi permission backend va demo data that.
